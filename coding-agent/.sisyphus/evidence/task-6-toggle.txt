### 2. Function Implementation (GREEN)
Added `use_pipeline()` function to `src/coding_agent/core/config.py`:
- Accepts optional `override: bool | None` parameter
- Returns True if override is not None (takes precedence)
- Returns True if USE_PIPELINE env var is "1", "true", or "yes" (case-insensitive, whitespace-stripped)
- Returns False by default (old path remains default during transition)

### 3. Key Implementation Details
- Function signature: `def use_pipeline(override: bool | None = None) -> bool:`
- Env var matching: `os.environ.get("USE_PIPELINE", "").strip() in ("1", "true", "yes")`
- Default behavior: False (preserves old AgentLoop path by default)
- Override parameter enables CLI flag support (T13) without modifying this function
