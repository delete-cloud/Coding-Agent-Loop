# Section 2A — HTTP Session Tape Continuity

> **Status**: Draft
> **Date**: 2026-04-13

> **Scope**: `agentkit/tape/store.py` (ForkTapeStore fix), `coding_agent/app.py`,

## Problem Statement

Current `SessionManager.run_agent()` creates a fresh `Tape()` on every
HTTP turn. Conversation history is lost between turns, between process
restarts, and after LRU eviction. This must be fixed before checkpoint
integration (Section 2B) can be meaningful.

---

## Prerequisite: Stable Tape Identity (agentkit change)

### The bug

Current `ForkTapeStore.commit()` persists delta entries to `fork.tape_id`,
which is a new UUID generated by `Tape.fork()` each turn.

```
Turn 1: ctx.tape (id="aaa") → fork (id="bbb") → save("bbb", delta)
Turn 2: ctx.tape (id="bbb") → fork (id="ccc") → save("ccc", delta)
```

Result: no single tape_id contains the full conversation history.
`tape_store.load("aaa")` returns nothing. `tape_store.load("ccc")` returns
only turn 2's delta.

### The fix

`ForkTapeStore.commit()` writes delta to the **base tape's id**, not the
fork's id.

```python
# agentkit/tape/store.py

class ForkTapeStore:
    def __init__(self, backing: TapeStore) -> None:
        self._backing = backing
        self._active: dict[str, Tape] = {}
        self._finalized: set[str] = set()
        self._base_lengths: dict[str, int] = {}
        self._base_tape_ids: dict[str, str] = {}     # NEW

    def begin(self, tape: Tape) -> Tape:
        forked = tape.fork()
        self._active[forked.tape_id] = forked
        self._base_lengths[forked.tape_id] = len(tape)
        self._base_tape_ids[forked.tape_id] = tape.tape_id   # NEW
        return forked

    async def commit(self, fork: Tape) -> None:
        if fork.tape_id in self._finalized:
            raise ValueError(f"tape '{fork.tape_id}' already finalized")
        self._finalized.add(fork.tape_id)
        self._active.pop(fork.tape_id, None)
        delta = fork.to_list()[self._base_lengths[fork.tape_id]:]
        stable_id = self._base_tape_ids[fork.tape_id]         # CHANGED
        await self._backing.save(stable_id, delta)             # CHANGED
```

### Post-commit identity rebind

The fix above is necessary but **not sufficient**. After
`ForkTapeStore.commit()` succeeds, `Pipeline.run_turn()` leaves
`ctx.tape` pointing to the fork (whose `tape_id` is a transient UUID).
The next turn's `begin(ctx.tape)` would use that transient id as the
new base — breaking stable identity again.

**Fix**: `commit()` returns the stable base tape id. `Pipeline.run_turn()`
rebinds `ctx.tape` identity after successful commit.

```python
# agentkit/tape/store.py — commit returns stable id

async def commit(self, fork: Tape) -> str:
    """Persist delta and return the stable base tape_id."""
    if fork.tape_id in self._finalized:
        raise ValueError(f"tape '{fork.tape_id}' already finalized")
    stable_id = self._base_tape_ids[fork.tape_id]
    delta = fork.to_list()[self._base_lengths[fork.tape_id]:]
    await self._backing.save(stable_id, delta)
    # Finalize bookkeeping AFTER successful save (not before)
    self._finalized.add(fork.tape_id)
    self._active.pop(fork.tape_id, None)
    self._base_lengths.pop(fork.tape_id, None)
    self._base_tape_ids.pop(fork.tape_id, None)
    return stable_id
```

```python
# agentkit/runtime/pipeline.py — rebind after commit

if fork is not None:
    stable_id = await ctx.storage.commit(fork)
    ctx.tape._tape_id = stable_id   # rebind to stable identity
```

**Key changes from original proposal:**

1. `_finalized.add()` moves **after** `backing.save()` — prevents
   masking save failures as "already finalized" in rollback
2. Bookkeeping dicts (`_base_lengths`, `_base_tape_ids`) are cleaned
   up after commit, preventing unbounded growth
3. `commit()` returns `str` (the stable id) instead of `None`
4. Pipeline rebinds `ctx.tape` identity so the next turn's
   `begin()` uses the correct stable id

### Semantic change

| Before | After |
|--------|-------|
| Each fork's delta saved under fork's own UUID | Each fork's delta appended under base tape's stable UUID |
| No single id contains full history | `tape_store.load(base_tape_id)` returns complete history |
| Fork id = persistence id | Fork id = transaction-scoped working id only |
| `ctx.tape.tape_id` drifts each turn | `ctx.tape.tape_id` rebinds to stable id after commit |
| `_finalized` set before save | `_finalized` set after save succeeds (fail-safe) |

### Impact

- **Production code**: no code path depends on "persisted id == fork.tape_id"
- **`Tape.fork()` / `parent_id`**: unchanged; `parent_id` is not used in load/save
- **Tests**: `tests/agentkit/tape/test_store.py` must update from
  `load(fork.tape_id)` to `load(base_tape_id)`
- **TapeStore protocol**: unchanged (`save(tape_id, entries)` signature is stable)
- **Subagent**: unaffected — subagent creates a child pipeline with a fresh
  `tape_fork`, which becomes its own stable base id

## Design: Session Continuity

### Two paths

| Path | When | What happens |
|------|------|-------------|
| Hot | Same process, session in cache | Reuse pipeline + ctx + adapter. Tape lives in memory. |
| Cold | Process restart / pod drift / LRU eviction | Load entries from TapeStore → rebuild Tape → create_agent(tape=...) → mount |

### Recovery fidelity

Cold restore provides **degraded continuity**, not full runtime continuity.

| Recoverable | Not recoverable |
|-------------|-----------------|
| Conversation history (all tape entries) | Shell cwd / env live state |
| Window position (handoff anchors) | MCP connection state |
| Topic anchors (TopicPlugin can rebuild from tape) | Accumulated metrics counters |
| Memory records (persisted in tape or JSONL) | Plugin-internal caches |

This is acceptable. Cold restore re-establishes conversation context for
the LLM. Plugins that need their runtime state will re-initialize through
normal mount. The user may notice the shell session is fresh, but the
conversation continues coherently.

## Change 3: SessionManager hot/cold paths

### Overview

```python
class SessionManager:
    def __init__(
        self,
        store: SessionStore | None = None,
        tape_store: TapeStore | None = None,
    ):
        self._store = store or create_session_store()
        self._tape_store = tape_store   # may be None for v1; see TapeStore section
        self._cache: SessionCache = SessionCache(max_sessions=50)
        self._approval_stores: dict[str, ApprovalStore] = {}
        self._lock = asyncio.Lock()

    async def run_agent(self, session_id: str, prompt: str) -> None:
        session = self.get_session(session_id)
        session.last_activity = datetime.now()
        session.turn_in_progress = True
        self._persist_session(session)

        try:
            if session.adapter is not None:
                await self._run_hot(session, prompt)
            else:
                await self._run_cold(session, prompt)
        except Exception as exc:
            # ...existing error handling (wire error message + TurnEnd)...
            raise
        finally:
            session.turn_in_progress = False
            self._persist_session(session)
```

### Hot path: reuse adapter

```python
async def _run_hot(self, session: Session, prompt: str) -> None:
    """Hot path: pipeline, ctx, and adapter already live in memory."""
    await session.adapter.run_turn(prompt)
```

Key point: **adapter AND consumer are reused**, not recreated. Consumer is
session-scoped (bound to `session.wire`), so it survives across turns
without rebinding. This avoids re-triggering `pipeline.mount(ctx)` which
is not idempotent for all plugins (MCP server init, shell state setup,
memory record loading).

> **Future note**: if transport ever becomes request-scoped rather than
> session-scoped, add a small public `PipelineAdapter.set_consumer()` method
> at that point. Not needed for v1.

### Cold path: rebuild from TapeStore

```python
async def _run_cold(self, session: Session, prompt: str) -> None:
    """Cold path: rebuild pipeline from persisted tape."""
    tape = await self._restore_tape(session.tape_id)

    pipeline, ctx = create_agent(
        workspace_root=session.repo_path,
        model_override=session.model_name,
        provider_override=session.provider_name,
        base_url_override=session.base_url,
        max_steps_override=session.max_steps,
        approval_mode_override=_approval_mode_str(session.approval_policy),
        session_id_override=session.id,
        tape=tape,
    )

    if session.provider is not None:
        llm_plugin = pipeline._registry.get("llm_provider")
        llm_plugin._instance = session.provider

    consumer = self._make_consumer(session)
    ctx.config["wire_consumer"] = consumer
    ctx.config["agent_id"] = ""

    adapter = PipelineAdapter(pipeline=pipeline, ctx=ctx, consumer=consumer)

    # Cache runtime objects for subsequent hot turns
    session.pipeline = pipeline
    session.pipeline_ctx = ctx
    session.consumer = consumer
    session.adapter = adapter
    session.tape_id = ctx.tape.tape_id    # stable id from here on
    self._persist_session(session)         # persist tape_id

    await adapter.run_turn(prompt)

async def _restore_tape(self, tape_id: str | None) -> Tape | None:
    """Load tape from TapeStore. Returns empty Tape with stable id if
    no entries found (preserves identity for subsequent commits).

    Raises on real I/O / corruption errors — do not silently mask
    failures as "fresh start", which would discard conversation
    history without the caller knowing.
    """
    if tape_id is None or self._tape_store is None:
        return None
    entries = await self._tape_store.load(tape_id)
    if not entries:
        return Tape(tape_id=tape_id)   # preserve stable id
    return Tape.from_list(entries, tape_id=tape_id)
```

### First turn of a new session

When `session.tape_id` is `None` and `session.adapter` is `None`:
- `_run_cold()` is called
- `_restore_tape(None)` returns `None`
- `create_agent(tape=None)` → `Tape()` with a fresh UUID
- `session.tape_id = ctx.tape.tape_id` — this is the stable id for the session's lifetime
- Subsequent turns via `ForkTapeStore.commit()` append to this stable id

## Change 4: SessionCache (LRU eviction)

```python
from collections import OrderedDict

class SessionCache:
    """LRU cache for hot sessions.

    Eviction is safe: tape is durably persisted every turn via
    ForkTapeStore.commit() to the session's stable tape_id.
    Evicted sessions lose in-memory runtime objects (pipeline, ctx,
    adapter) but retain full conversation history in TapeStore.
    Next access triggers cold restore.
    """

    def __init__(self, max_sessions: int = 50):
        self._cache: OrderedDict[str, Session] = OrderedDict()
        self._max = max_sessions

    def get(self, session_id: str) -> Session | None:
        if session_id in self._cache:
            self._cache.move_to_end(session_id)
            return self._cache[session_id]
        return None

    def put(self, session: Session) -> Session | None:
        """Add/update session. Returns evicted Session if cache was full."""
        evicted = None
        if session.id in self._cache:
            self._cache.move_to_end(session.id)
        elif len(self._cache) >= self._max:
            _, evicted = self._cache.popitem(last=False)
        self._cache[session.id] = session
        return evicted

    def remove(self, session_id: str) -> None:
        self._cache.pop(session_id, None)
```

Eviction clears `session.pipeline`, `session.pipeline_ctx`, `session.adapter`
(they become eligible for GC). `session.tape_id` survives in the session store.

## TapeStore Ownership

### Principle

SessionManager depends on a **continuity TapeStore** built from the same
storage backend configuration as StoragePlugin. It is NOT an independent
second storage authority.

### V1 constraint

Section 2A v1 supports **JSONL backend only**. Both SessionManager and
StoragePlugin read/write `{data_dir}/tapes/`.

```python
class SessionManager:
    def __init__(
        self,
        store: SessionStore | None = None,
        tape_store: TapeStore | None = None,

        if self._tape_store is None:
            data_dir = Path(os.environ.get("AGENT_DATA_DIR", "./data"))
```

## SessionLock

### Layered approach

| Version | Scope | Implementation |
|---------|-------|---------------|
| v1 | Single process | `asyncio.Lock` per session (already exists as `self._lock`) |
| v2 | Multi-process / multi-pod | PG advisory lock or SQLite lock table |

### V1: per-session asyncio.Lock

Current `SessionManager` has a single `self._lock` for all operations.
Replace with per-session granularity:

```python
class SessionManager:
    def __init__(self, ...):
        ...
        self._session_locks: dict[str, asyncio.Lock] = {}

    def _get_lock(self, session_id: str) -> asyncio.Lock:
        if session_id not in self._session_locks:
            self._session_locks[session_id] = asyncio.Lock()
        return self._session_locks[session_id]

    async def run_agent(self, session_id: str, prompt: str) -> None:
        lock = self._get_lock(session_id)
        if lock.locked():
            raise SessionBusy(session_id)
        async with lock:
            ...existing turn logic...
```

### V2 protocol (for reference, not implemented in 2A)

```python
class SessionLock(Protocol):
    async def acquire(self, session_id: str, ttl: int = 30) -> bool: ...
    async def release(self, session_id: str) -> None: ...

# Implementations: PGSessionLock (advisory lock), SQLiteSessionLock (lock table)
```

## Data Flow

### Turn N (hot path)

```
run_agent(session_id, prompt)
  │
  ├─ session_lock.acquire(session_id)
  ├─ cache.get(session_id) → Session (has pipeline, ctx, adapter, consumer)
  ├─ adapter.run_turn(prompt)
  │   ├─ tape.append(user_message)
  │   ├─ Pipeline.run_turn(ctx)
  │   │   ├─ ForkTapeStore.begin(ctx.tape)    ← base_tape_id = session.tape_id
  │   │   ├─ ...LLM runs...
  │   │   └─ ForkTapeStore.commit(fork)       ← save(session.tape_id, delta) ✓
  │   └─ TurnEnd on wire
  ├─ persist_session(metadata)
  └─ session_lock.release(session_id)
```

### Cold restore (after restart / eviction)

```
run_agent(session_id, prompt)
  │
  ├─ session_lock.acquire(session_id)
  ├─ cache.get(session_id) → None
  ├─ store.load(session_id) → metadata { tape_id: "aaa" }
  ├─ tape_store.load("aaa") → full entry history
  ├─ Tape.from_list(entries, tape_id="aaa")
  ├─ create_agent(tape=restored_tape, ...) → pipeline, ctx
  ├─ Pipeline.mount(ctx) → plugins initialize (degraded: no shell/MCP state)
  ├─ cache.put(session) → hot for subsequent turns
  ├─ adapter.run_turn(prompt) → same as hot path from here
  └─ session_lock.release(session_id)
```
